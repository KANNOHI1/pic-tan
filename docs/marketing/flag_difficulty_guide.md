# Pic-tan 国旗テーマ難易度設計と親向け説明文 (PT-068)

> **作成日**: 2026-03-05  
> **対象**: マーケティング・カスタマーサポート・プロダクトチーム  
> **目的**: 国旗テーマにおいて、なぜ「Core（5〜6歳向け）が国名」で、「Challenge（7〜8歳向け）が首都」なのか、その教育的意図を保護者に納得感のある形で説明するためのガイドライン。

## 🎯 難易度設計の背景（年齢・認知適合性）

Pic-tanのターゲットである4〜8歳の認知発達段階において、地理や世界への関心は以下のように推移します。

1. **Easy (4〜5歳想定)**
   - 世界の概念が未熟なため、国旗テーマ自体を「非表示」としています。身近な動物や果物に集中させます。
2. **Core (5〜6歳想定)**
   - 「自分たちの住む国以外にも、別の国がある」という概念が芽生える時期です。
   - カラフルな国旗の「デザイン」から入り、まずは**「国名（Country）」**という大きな括りを結びつけることに絞ります。
3. **Challenge (7〜8歳想定)**
   - 小学校に入学し、物事の階層構造（国の中に都市がある）を理解できるようになります。
   - そのため、国旗から直接**「首都（Capital）」**を導き出す一段高い認知負荷を与え、知的好奇心を刺激します（補足として国名も表示します）。

## 💬 親向け説明用コピー（FAQ・ストア・LP用）

この設計思想を保護者に伝える際は、以下の文案を使用します。（誇大表現を排除し、発達段階に合わせた適切な学習であることを強調します）

### 【メイン文案】
**「お子さまの成長に合わせた、ムリのないステップアップ」**

Pic-tanの国旗カードは、年齢別の認知発達に合わせて出題内容が変わります。
5〜6歳向けの「Core」レベルでは、まずはカラフルな国旗のデザインと**「国名」**を結びつけ、世界への興味を育みます。
7〜8歳向けの「Challenge」レベルになると、国旗から**「首都」**を答える一段階上のクイズに変化。国の中に都市があるという階層理解を促し、知的好奇心を刺激します。
