# 📱 Pic-tan: 審査前プレフライト・コピーチェックリスト

> **作成日**: 2026-03-05  
> **対象**: マーケティング・法務およびApp Store申請担当（リリース運用者向け）  
> **目的**: App Store Connectへの登録直前、および各アップデート時に、メタデータ（タイトル、説明文、スクリーンショット）内の表現が「法務的・プラットフォーム的リスク」を犯していないか、1ページでセルフチェックするためのシート。

---

## 🟢 1. 許可される表現（Allowed Wording）

これらの表現は Evidence Brief（科学的根拠）に準拠しており、「推奨」として使ってよい表現です。

- [ ] 学習の **「プロセス」** をサポートする旨
  - 例：「記憶に残りやすくなります」
  - 例：「ことばの引き出しが作られやすくなります」
- [ ] 学習の **「習慣化・接触の工夫」** を謳う旨
  - 例：「無理なく毎日続く」
  - 例：「1日3分の積み重ねが定着の鍵」
- [ ] 再想起（思い出すこと）の **「訓練・機能」** を謳う旨
  - 例：「思い出す力をサポートします」
  - 例：「振り返ることで、思い出す行動そのものを後押しします」
- [ ] 環境やツールの **「安全性」** について
  - 例：「広告のない静かな画面」
  - 例：「目の前のことばに集中できます」

## 🔴 2. 禁止・リスク表現（Forbidden Wording）

これらの表現は使用を控えてください。ストアからのリジェクト（Reject）、法務リスク（景品表示法違反やFTC規制違反）、ユーザー（親）からの炎上の原因となります。もしメタデータに記載がある場合は即座に修正が必要です。

### A. 効果の「絶対的保証」や過度な断定
- [ ] 「必ず覚えられる」
- [ ] 「絶対に定着する」
- [ ] 「ネイティブ並みになる」
- [ ] 「バイリンガルになれる」
- [ ] 「成績が上がる」 / 「英検に合格できる」
- [ ] （英語限定） "Guaranteed fluency", "Proven to work"

### B. 科学的根拠の「誇大解釈・知能向上」
- [ ] 「IQが上がる」
- [ ] 「右脳を活性化する」
- [ ] 「天才になる」
- [ ] 「脳科学アプリ」といった過剰な表記（「科学的アプローチに基づき〜」等は可）

### C. 短時間・無駄に関する「過激な比較表現」
- [ ] 「たった3分で英語がペラペラに」
- [ ] 「長時間の学習は無意味です」
- [ ] 他の英語学習アプリやメソッドを名指しで下げるような比較広告

### D. 医療・障害へのアプローチ
- [ ] 「ADHDに効果がある」
- [ ] 発達障害や学習障害の「改善・治療」を目的とするかのような表現

### E. 親の責任放棄を示唆する表現
- [ ] 「親は何もしなくていい」
- [ ] 「スマホに子守りを任せられる」
- [ ] （日本市場特有の）「アプリが全部やってくれる」といった、親子コミュニケーション（共同注意）の価値を落とす表現

## 📝 3. 運用フロー

1. App Store Connect の `プロモーションテキスト`, `概要(Description)`, `サブタイトル` の原稿を用意。
2. スクリーンショットのバナーコピー原稿を用意。
3. このチェックリストの「🔴 2. 禁止・リスク表現」に該当する文言がないかを指差し確認。
4. 全てクリアした場合のみ、審査承認（Submit for Review）を実行。
