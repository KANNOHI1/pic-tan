# PRD: ピクたん / Pic-Tan

**Version:** 1.0
**Last Updated:** 2026-03-16
**Status:** Draft

---

## 1. Summary

ピクたんは、親子で毎日3分間の英単語学習を楽しむiOSアプリである。かわいい画像を使った直感的な学習体験を提供し、子どもが英語に自然に親しめる環境を作る。フリーミアムモデルで無料提供し、一度限りの課金（JPY 480）でフルアクセスを解放する。

## 3. Background

### このプロジェクトは何か

子どもの英語学習市場には、ゲーム型・動画型・フラッシュカード型など多数のアプリが存在する。しかし「親子で一緒に3分間」という短時間ルーティンに特化したアプリは少ない。

### なぜ今か

- 生成AIによる画像生成技術が実用レベルに達し、高品質な学習用イラストを低コストで量産できる
- iOSのSwiftUIが成熟し、個人開発者でも高品質なアプリを作れる環境が整っている
- 幼児英語教育への関心が高まり、App Storeでの需要が増している

---

## 4. Objective

### 目的

子どもが楽しく英単語に触れる習慣を作り、親が安心して使えるアプリを届ける。

### 会社・ユーザーへの価値

- **ユーザー価値：** 広告なし・安全・かわいい画像で子どもが喜ぶ。親の負担が少ない（3分で完結）
- **事業価値：** 一度限り課金モデルで継続収益。App Store評価の積み上げ

### Key Results（成功指標）

| 指標 | 目標 |
|---|---|
| App Store初月DL数 | 500 DL |
| 課金転換率 | 3%以上 |
| 平均セッション継続日数 | 7日以上 |
| App Storeレーティング | 4.5以上 |
| 年齢レーティング | 4+ 維持 |

---

## 5. Market Segment

### 誰のために作るか

**プライマリユーザー：** 2〜6歳の子ども
**プライマリバイヤー：** 親・保護者（iPhoneユーザー、30代前後）

### 制約

- 年齢レーティング4+のため、広告・外部リンク・SNS機能は禁止
- 子どもが一人でも操作できるシンプルなUXが必須
- 日本語ファーストでリリース、将来的に多言語対応

### 対象外

- 小学生以上（難易度が合わない）
- Androidユーザー（現フェーズは対象外）

## 6. Value Proposition

### 顧客が解決したい問題（Jobs）

- 子どもに英語を教えたいが、自分に英語力がなくても使えるものが欲しい
- 長続きする学習習慣を作りたい
- 子どもが安全に使えるアプリを選びたい

### ピクたんが提供するもの（Gains）

- かわいいイラストで子どもが自分から「やりたい」と言う
- 3分で終わる設計で、毎日続けやすい
- 広告なし・課金誘導なしで親が安心

### 避けられる問題（Pains）

- 動画やゲームに夢中になりすぎて学習にならない問題を回避
- 不適切な広告が表示されるリスクをゼロに

### 競合との差別化

| 比較軸 | ピクたん | 競合アプリ一般 |
|---|---|---|
| 対象年齢 | 2〜6歳に特化 | 幅広い |
| 広告 | なし | あり（無料版） |
| 学習時間 | 3分設計 | 制限なし |
| 親子体験 | 親子で一緒に設計 | 子ども単独が多い |

## 7. Solution

### 7.1 UX・画面フロー

```
起動 → 年齢帯選択（初回のみ）→ ホーム画面 → テーマ選択 → 学習セッション（カード表示）→ 完了画面
```

- 年齢帯によって表示テーマ・難易度を自動ゲーティング
- タップのみで操作完結（文字入力なし）

### 7.2 主要機能

| 機能 | 説明 | フェーズ |
|---|---|---|
| 画像フラッシュカード | 英単語+日本語+かわいいイラスト | MVP |
| テーマ別学習 | 動物・色・果物・国旗など | MVP |
| 年齢帯ゲーティング | 年齢に応じた難易度切替 | MVP |
| 学習完了画面 | 達成感を演出 | MVP |
| フリーミアム解放 | 一度限り課金でフルアクセス | MVP |
| ストリーク表示 | 連続学習日数の可視化 | v1.1以降 |
| 保護者向け説明画面 | なぜ画像ファースト？なぜ3分？ | v1.1以降 |
| サウンド・アニメーション | 子ども向け演出強化 | v1.1以降 |

### 7.3 技術

- **フロントエンド：** SwiftUI（iOS）
- **コアロジック：** packages/core（TypeScript）
- **画像：** AI生成イラスト（Imagen API）、PNG形式
- **コンテンツ：** JSONバンドル（app resourcesに内包）
- **課金：** StoreKit（一度限り課金）

### 7.4 前提・未検証仮説

- 親が「3分」という時間制限に価値を感じる（未検証）
- 子どもがイラストの質に反応する（部分検証済み）
- JPY 480の価格が転換率に最適（未検証）
- Xcode/macOS環境でのビルドが通る（PT-011で検証中）
