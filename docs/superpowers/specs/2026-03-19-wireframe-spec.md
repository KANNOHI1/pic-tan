```

│                                     │
│  ①  タイトル                        │  h: ~60px
│     「なかよしパートナーをえらんでね」 │
│     ~~~~~~~~~~~~~~~~~~~~            │
│                                     │
│  ②  キャラ横並びスクロール           │  h: ~320px
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐      │
│  │🐰│ │🐱│ │🐶│ │🐧│ │🐦│  →→  │  各80px×80px
│  │  │ │  │ │  │ │  │ │  │      │
│  └──┘ └──┘ └──┘ └──┘ └──┘      │
│  ルル チャイ ポチ  ペペ  ソラ        │  名前ラベル
│                                     │
│  ③  吹き出し（タップ時に表示）       │  h: ~100px
│  ┌─────────────────────────────┐   │
│  │ 「〜だもん！よろしくね！」   ▲  │   │  → タップしたキャラの台詞
│  └─────────────────────────────┘   │
│                                     │
│  ④  [     けってい！     ]          │  h: 64px, 全幅
│                                     │
└─────────────────────────────────────┘
```

### 動作注記
| トリガー | 動作 |
|---|---|
| キャラをタップ | そのキャラを選択状態に（枠でハイライト）＋吹き出しで自己紹介 |
| 選択なしで「けってい！」 | ボタン非活性（タップ不可）|
| 選択後「けってい！」 | localStorage に `pictan_partner` 保存 → ホーム画面へ |
| 2回目以降の起動 | この画面をスキップ、ホーム画面へ直行 |

### 状態
- **未選択**: 吹き出し非表示、「けってい！」ボタングレーアウト
- **選択済み**: 選択キャラにリング枠、吹き出し表示、ボタン活性

## レスポンシブ注記

- 基準: 375px × 667px（iPhone SE相当）
- 大画面（414px以上）: 余白を左右に追加、中央揃え
- 小画面（320px）: フォントサイズを一段小さく、ボタン高さ維持

## アクセシビリティ注記

- 全タップ可能要素: 最低 44×44px タップ領域
- カラーのみの情報伝達禁止: テキストまたはアイコンを必ず併用
- 評価ボタン: テキスト＋絵文字で意味を伝達（色だけに依存しない）
