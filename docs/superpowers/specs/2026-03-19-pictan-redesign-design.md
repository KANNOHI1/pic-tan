# Pic-tan リデザイン設計書

**日付**: 2026-03-19
**対象**: apps/web（Webプレビュー）
**ステータス**: 承認済み

---

## 1. コンセプト

### ビジョン
「勉強した動物が自分のまちに住み着く」体験を通じて、英語学習と街づくりが一体化したナラティブループを作る。

### 感情トーン
- **ワクワク**（ゲーム感覚・次が気になる）
- **温かい**（絵本・どうぶつの森の世界観）

### 参照デザイン
どうぶつの森（温かいパステル＋ポップなアクセント・街・自然の世界観）

---

## 2. コアナラティブループ

```
初回起動
  → パートナー選択（5匹から1匹）
  → 荒れ地＋パートナーの小屋だけの場所へ

毎日のセッション
  → パートナーが「きょうは○○にあいにいこう！」と誘う
  → カード学習（2段階めくり＋2択評価）
  → 完了：今日会った動物との仲良し度が上がる

蓄積
  → 仲良し度が一定以上 → 動物がまちに引っ越してくる
  → 住民が増えるほどまちが発展
  → 全語彙マスター ≒ にぎやかな可愛いまち
```

---

## 3. 画面構成（5画面）

### [1] キャラクター選択画面（初回のみ）
- 5匹が横並び（スクロール可）
- タップすると吹き出しで自己紹介
- 「けってい！」ボタンで確定

### [2] ホーム画面（まちの全景）
- **上部（約55%）**: まちのイラスト（発展段階で変化）。パートナーキャラクターはまちの絵の中を動く
- **中部（約15%）**: パートナーの吹き出し「きょうは○○にあいにいこう！」
- **下部（約25%）**: 大きな「いく！」ボタン1つのみ（高さ64px以上、全幅）
- **右上（固定）**: 保護者ボタン（小さく、44px×44pxタップ領域確保）
- モバイル基準（375px幅・667px高）で縦に収まるよう設計する

### [3] カード学習画面
- 上：進捗ドット（全カード枚数）
- 中：大きな絵カード（画面の約50%）
- 下：タップヒント
  - 1タップ目 → 英語（大きく）
  - 2タップ目 → 日本語（英語の下に）
  - 日本語表示後 → 評価ボタン登場
    - 「しってた！😊」
    - 「むずかしい😵」

### [4] 完了画面
- パートナー絵文字と今日会った動物絵文字を横並びで表示
- 両方をbounceアニメーション（揺らす）→ ハートが間に浮かぶ演出
- 「○○ちゃんとなかよしになったよ！」テキスト
- 仲良し度の変化（ハートアイコンが増える。例：♡♡♡ → ♡♡♡♡）
- 「またあした！」ボタン

### [5] 保護者パネル（オーバーレイ）
- 今日練習した単語一覧
- 「忘れかけているころにまた練習します（エビングハウス忘却曲線）」説明
- **次回復習予定**（単語ごとの復習日、例：「いぬ → 明日」「ねこ → 3日後」）
- まちの成長状況（住民一覧・仲良し度）
- 安全性バッジ（広告なし・オフライン・個人情報なし）

## 4. カード学習メカニクス

### セッションのカード枚数
- 1セッションあたり **8枚固定**
  - 新規カード：最大5枚
  - 復習カード（「むずかしい」判定済み）：残り枠を埋める

### 表示フロー
```
絵だけ表示（静止）
  → タップ → 英語登場（大きく中央）
  → タップ → 日本語追加（英語の下）
  → 「しってた！😊」「むずかしい😵」が現れる
  → 選択 → 次のカードへ
```

### スペースドリペティション（内部ロジック）
- 「しってた！」→ 次回は間隔を空けて出す（1日後 → 3日後 → 7日後）
- 「むずかしい」→ 次のセッションで優先的に出す
- 子どもには見えない（UIに出さない）
- 保護者パネルに「次回復習予定」として表示

### 仲良し度ロジック
```
初回「しってた！」→ 仲良し度 +2
初回「むずかしい」→ 仲良し度 +1
再会「しってた！」→ 仲良し度 +3
仲良し度 10以上 → まちへの引っ越し解放
引っ越し後：仲良し度の加算は継続、上限なし（まち内での活躍度として保持）
```

### データ永続化（localStorage）
```js
// パートナー選択（初回のみ）
localStorage.setItem('pictan_partner', 'rabbit') // rabbit/cat/dog/penguin/bird

// 仲良し度（動物ごと）
localStorage.setItem('pictan_friendships', JSON.stringify({
  dog: 3, cat: 12, apple: 0, ...
}))

// スペースドリペティション（次回復習日）
localStorage.setItem('pictan_next_review', JSON.stringify({
  dog: '2026-03-20', cat: '2026-03-22', ...
}))

// streak（既存キーを踏襲）
localStorage.setItem('pictan_streak', '5')
localStorage.setItem('pictan_streak_date', '2026-03-19')
```

起動時チェック：`pictan_partner` が存在 → ホーム画面へ。存在しない → キャラ選択画面へ。

## 7. ビジュアル言語

### カラーパレット
| 役割 | 色 | HEX |
|---|---|---|
| 背景 | 温かいクリーム | `#FFF8E7` |
| まちの草 | やわらかいグリーン | `#A8D5A2` |
| アクセント | テラコッタ | `#E8835A` |
| テキスト | チャコール | `#3D3227` |
| 成功・喜び | サニーイエロー | `#FFD166` |
| 仲良し度 | コーラルピンク | `#FF8FAB` |

### タイポグラフィ
- 日本語：Noto Sans JP（丸ゴシック）
- 英語：Nunito（丸みのある子ども向け）
- 英単語カード：48〜64px / 太字
- ひらがな：28〜32px

### 形状・質感
- 角丸：`border-radius: 16〜24px` を基本
- 影：薄いソフトシャドウのみ（float感）
- カード：白い厚みのあるカード風
- まちのイラスト：フラットイラスト（塗り絵感）

### アニメーション
| 対象 | 内容 | 時間 |
|---|---|---|
| カードめくり | ease-out | 150ms |
| キャラのしぐさ | ゆっくりbounce | ループ |
| 住民が来るとき | scale 0→1 ポンッ | 300ms |
| 完了演出 | ハート・星がふわっと舞う | 600ms |

## 9. アクセシビリティ

- すべてのタップ可能要素は最低 44px × 44px のタップ領域を確保
- カラーのみによる情報伝達を禁止（テキストまたはアイコンを必ず併用）
- `role`・`aria-label` は現行実装の慣習を踏襲
- 保護者がマウス操作する場面（PCブラウザ）を想定し、`:hover` スタイルを用意
